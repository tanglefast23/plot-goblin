# Design Context — Plot Goblin

Synthesized from `AGENTS.md`, `README.md`, and `docs/plot-goblin-style-guide.md` (source of truth). Update those first; this mirrors them for the impeccable design skills.

## Target Audience
Screenwriters and aspiring filmmakers with a messy movie idea who need to build a usable story spine (premise, protagonist, want, stakes, false belief, opposition, theme, beats, scenes) *before* writing pages. Comfortable with creative tools; not necessarily technical.

## Use Cases
- Guided setup: a step-by-step flow that turns a vague idea into a structured story spine.
- Rooms: editable markdown work surfaces (Premise, Characters, Theme, Beats, Scenes) for developing each part of the story.
- "Feed the Goblin": invoking AI suggestions and helpfully annoying follow-up questions.
- Export: bundle the whole project into one markdown file.

## Brand Personality / Tone
Funny, mischievous, **helpfully annoying** goblin editor. Confident, not busy. Mischievous, not gross. Useful, not ornamental. Poster-like, not dashboard-first. The goblin nudges and questions; it never silently rewrites and never insults the writer.

## Visual Direction (locked)
Pale green editorial poster. Flat mist background, dark swamp-green typography, simple goblin/page/quill silhouettes, rounded pill buttons, sparse navigation, bold uppercase headlines with extra-heavy emphasis words. Do NOT drift to parchment/rust/orange.

## Motion Constraints
- Reinterpret playful/"Duolingo-style" feedback as **crisp, confident motion** — use `ease-out-quart/quint/expo`, NEVER bounce or elastic (conflicts with "confident, not busy").
- `transform` + `opacity` only (GPU-friendly). Never animate layout properties.
- Always honor `prefers-reduced-motion`.
- Feedback 100–150ms, state changes 200–300ms, entrances 300–500ms. Exits ~75% of entrance.
- One signature moment beats scattered micro-interactions everywhere.

## Tech Constraints
Next.js 16 (App Router) + React 19, CSS Modules + a few globals tokens, localStorage persistence, no animation libraries currently installed (prefer CSS / Web Animations API over adding deps).
